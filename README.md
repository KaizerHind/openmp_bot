# openmp_bot
No official BOT.

## Install
```bash
git clone https://github.com/KaizerHind/openmp_bot.git
cd openmp_bot
git checkout neetoons
git submodule update --init
npm i 
```

## Create a .env file with your token
```
TOKEN = your_token 
```

## Deploy 
```
node deploy-commands.js
node index.js
```
